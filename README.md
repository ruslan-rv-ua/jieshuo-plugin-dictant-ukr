# "Диктант" — плагін Jieshuo для зручного надиктовування тексту

Плагін для скрінрідера [Jieshuo](https://jieshuo.app/) на Android. Дозволяє надиктовувати текст у будь-яке поле редагування з автоматичним форматуванням пунктуації та регістру.

## Встановлення

1. Завантаж `.ppk`-файл на пристрій.
2. У Jieshuo: Меню → Плагіни → Встановити → обери файл.

## Використання

Встановіть фокус на поле редагування та активуйте плагін. Почуєте вібрацію — починайте диктувати.

Якщо фокус не на полі редагування, ви почуєте "не в редакторі" і плагін зупиниться.

## Крок за кроком

Текст можна надиктовувати у кілька кроків: кожен новий надиктований текст додається до редактора, весь текст форматується з урахуванням пунктуації.

Приклад:

1. Надиктували `зараз` → у редакторі: `зараз`
2. Надиктували `сонячно` → у редакторі: `зараз сонячно`
3. Надиктували `крапка` → у редакторі: `Зараз сонячно.`
4. Треба переписати наново. Надиктували:
    ```
    диктант зараз у селі з великої великі з великої плоскогубці сонячно знак оклику
    ```
    У редакторі з'явиться:
    ```
    Зараз у селі Великі Плоскогубці сонячно!
    ```
    де `диктант` — очищення редактора, а `з великої` перед кожним словом — команда написати його з великої літери.

## Очистити редактор

Скажіть "диктант" на початку голосового набору, щоб видалити весь попередній текст. Корисно, якщо передумали або в полі є плейсхолдер (наприклад, "Пошук").

## Команди та пунктуація

Пунктуація:

| Команда | Символ |
|---|---|
| крапка | . |
| кома | , |
| двокрапка | : |
| знак питання | ? |
| знак оклику | ! |
| тире | — |
| ліва дужка | ( |
| права дужка | ) |

Команди:

| Команда | Дія |
|---|---|
| диктант | видалити весь текст із редактора |
| абзац | новий рядок |
| з великої | наступне слово — з великої літери |
